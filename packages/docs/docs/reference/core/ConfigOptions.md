## ConfigOptions

A central place for enumerating the configurable options of our GoldfinchConfig contract

### Numbers

```solidity
enum Numbers {
  TransactionLimit,
  TotalFundsLimit,
  MaxUnderwriterLimit,
  ReserveDenominator,
  WithdrawFeeDenominator,
  LatenessGracePeriodInDays,
  LatenessMaxDays,
  DrawdownPeriodInSeconds,
  TransferRestrictionPeriodInDays,
  LeverageRatio
}
```
